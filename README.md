# The DJI Multitool

This is a Work In Progress, the goal is to create an easily extendible web app for general DUML commands, including FCC unlocking, DroneID Disabling, parameter modification and more!

## Installation
```bash
# Install dependencies
npm install

# Build the app
npx gulp

# Run the dev server
npx gulp dev
```
